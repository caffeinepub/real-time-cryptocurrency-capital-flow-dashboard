# Guia de Construção do APK Android

Este guia explica como empacotar o Dashboard de Fluxo de Capital Cripto como um aplicativo Android nativo usando Capacitor.

## Pré-requisitos

1. **Node.js e npm** instalados
2. **Android Studio** instalado com Android SDK
3. **Java JDK 11 ou superior** instalado
4. Aplicação web já construída e funcionando
